# Crud_Techmanager
Projeto feito em BackEnd com o intuito da criação de um sistema CRUD para uma empresa de desenvolvimento

Autores: Elionay Ramalho - 03345778
Vitor Henrique Duarte Santiago - 03346242
Eduarda Maquiné dos Santos - 03348708

TechManage CRUD API RESTful para gerenciamento de usuários, projetos e tarefas usando Node.js, Express, Nodemon e Sequelize com MySQL.

Tecnologias Node.js Express Sequelize MySQL Dotenv Pré-requisitos Node.js instalado MySQL rodando (ex: Laragon, XAMPP)

Instalação e Comandos para Inicialização

Npm install - pastas node_modules
npm run dev - Iniciar servidor


Rotas principais: 
/users
/projects
/tasks


Para teste, use ferramentas de testes, como a extensão do Thunder Client no VS Code.
